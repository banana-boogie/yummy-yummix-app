# Thermomix Assets

This directory contains all the image assets used for the Thermomix cooking parameters display.

## Folder Structure

```
assets/images/thermomix/
├── temperature/      # Temperature-related images (both C and F)
├── speed/            # Speed-related images (both forward and reverse)
└── time/             # Time-related images
```

## Naming Conventions

### Temperature Icons
Temperature icons use a specific naming pattern:

- **Celsius**: `temp-c-{value}.png` (e.g., `temp-c-37.png`, `temp-c-100.png`)
- **Fahrenheit**: `temp-f-{value}.png` (e.g., `temp-f-100.png`, `temp-f-250.png`)
- **Special**: `temp-c-varoma.png` and `temp-f-varoma.png` (one for each unit)
- **Default**: `temp-c-default.png` and `temp-f-default.png` (used when no temperature is specified)

### Speed Icons
Speed icons use a consistent naming pattern:

- **Forward Speed**: `speed-{value}.png` (e.g., `speed-1.png`, `speed-10.png`)
- **Reverse Speed**: `speed-reverse-{value}.png` (e.g., `speed-reverse-1.png`)
- **Special**: `speed-spoon.png` and `speed-reverse-spoon.png`
- **Default**: `speed-default.png` and `speed-reverse.png`

Speed ranges don't have specific images - they use the default speed image with a text overlay.

### Time Icons
- **Time circle**: `time-default.png` (the background for time display)
